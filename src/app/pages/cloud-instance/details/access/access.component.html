Recovery Console
Utilize the Recovery Console if you need to access the recovery ISO or if you're unable to connect to your instance via the Instance Console. To access the Recovery Console, password authentication must be enabled. If needed, you can reset your root password below.
Reset root password
Clicking "Reset Root Password" below will promptly power down your instance and assign a new root password.
You will receive the new root password via email within a few minutes. If the email doesn't come through or the new password fails to work, consider using the recovery environment.
