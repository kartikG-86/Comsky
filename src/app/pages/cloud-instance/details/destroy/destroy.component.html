Destroy Instance
This action is irreversible. We will permanently delete your instance, and all data associated with the instance will be wiped and cannot be recovered.
Rebuild Droplet
This will reconstruct your instance using the specified image and its original configuration settings. The rebuild process will erase all data currently on this instance, so be sure to back up anything you wish to retain.
Rebuilds are restricted to base images and snapshots that belong to the same OS family as the instance.
